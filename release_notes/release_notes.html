Censys Release Notes - Published by Splunk June 21, 2022

Version 2.2.0 - Released June 21, 2022
  • Updated the App logo and Product Vendor
Version 2.1.13 - Released January 21, 2022
  • Marked the app as FIPS Compliant [PAPP-22665]
Version 2.1.11 - Released November 30, 2021
  • Updated endpoints to support Censys 2.0 API [PAPP-20791]
  • Updated logo to latest version [PAPP-20994]
Note: 'Query Domain' and 'Lookup Domain' actions are not supported in Censys 2.0 API
Version 2.0.8 - Released July 13, 2021
  • Compatibility changes for Python 3 support [PAPP-13953]
Version 1.0.30 - Released September 19, 2019
  • Changed the configuration parameters' data type to password
  • Fixed the documentation errors
Version 1.0.26 - Released April 24, 2019
  • Added three new actions
    • query domain
    • query IP
    • query certificate
Version 1.0.19 - Released February 07, 2018
  • App action views and Logo updates
Version 1.0.11 - Released July 27, 2017
  • 3.0 compatibility update release only
Version 1.0.10 - Released March 23, 2017
  • Fixed an error that allowed test connectivity to pass with bad credentials
  • Documentation updates and improvements
Version 1.0.7 - Released November 7, 2016
  • Initial release
